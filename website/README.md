# VaultLock — Landing Page

Marketing website for [VaultLock](https://github.com/anomalyco/vaultlock), a zero-knowledge, offline-first encrypted password manager.

Built with React 19, TypeScript, Vite, and Tailwind CSS 4.

## Scripts

| Command | Description |
|---|---|
| `pnpm dev` | Start Vite dev server on `http://localhost:5173` |
| `pnpm build` | Type-check and build for production |
| `pnpm preview` | Preview production build locally |
| `pnpm lint` | Run ESLint |

## Stack

- **React 19** + **TypeScript 6**
- **Vite 8** with fast HMR
- **Tailwind CSS 4** with `@tailwindcss/vite` plugin
- Custom scroll-triggered animations using `IntersectionObserver`

## Sections

- Hero — Product intro with CTA buttons and trust badges
- Features — 6 key features in a responsive grid
- Stats — Animated numbered stats with explanatory descriptions
- How It Works — 4-step setup walkthrough
- Tech Stack — Frontend, backend, security, and tooling
- Security — Encryption and auth specification table
- CTA — Download call-to-action with pulse glow
- Footer — Navigation links and copyright
